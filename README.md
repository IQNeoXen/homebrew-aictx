# homebrew-aictx

Homebrew tap for [aictx](https://github.com/IQNeoXen/aictx) â€” a context switcher for AI tool configurations, inspired by kubectx.

## What is aictx?

`aictx` lets you manage and switch between named AI provider configurations (endpoints, API keys, models) across multiple AI tools such as Claude Code CLI and Claude Code for VSCode. Think of it like `kubectx` but for your AI tooling.

## Add this tap

```sh
brew tap IQNeoXen/aictx
```

## Install aictx

```sh
brew install IQNeoXen/aictx/aictx
```

Or in a single command:

```sh
brew install IQNeoXen/aictx/aictx --no-quarantine
```
